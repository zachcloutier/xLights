Sequence statistics

Sometimes you just need to see the stats - and the Tools menu Export Effects has a bunch of stats about your sequence including:

  • A full effect by effect listing in table form including:
    • Start time
    • End time
    • Duration
    • Effect description - if you entered one
    • The model it is on
    • Files used by the effect
  • At the end a summary of your sequence including:
    • How often you used each effect type
    • The total time each effect type is used
    • The full list of files your sequence depends upon - images, videos etc
