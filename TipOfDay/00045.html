Package sequence

If you want to share a sequence with someone always use the tools menu package sequence method as this guarantees the package will include all the source files necessary to reproduce your sequence regardless of where on the computer they happen to be.

When using such a sequence there are some key things you need to do to ensure it works correctly and continues to work correctly once imported:

  • Either import the sequence directly from the zip file which ensures all file linkages are perfect but has the downside of cluttering up your folders and it makes it hard to review the sequence you are importing.
  • Or extract the zip file but:
    1. Extract the zip file into a sub folder of your show folder.
    2. Temporarily switch your show folder to the folder you extracted the sequence into.
    3. Check the layout is the layout you should be importing from.
    4. Load the sequence.
    5. Render the sequence and play it to check it all looks ok.
    6. Save the sequence - This is super super critical as it fixes all the file paths ready for the import.
    7. Switch back to your show folder,
    8. Import the sequence from the folder you extracted it into.

If sharing publicly you will want to exclude the audio file for copyright reasons. There is a setting option to do that. You may also want to exclude your presets while you are at it.
