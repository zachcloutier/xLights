Spiral Trees

Spiral trees like those created using tomato cages can be modelled in xLights using the Tree model.

The properies you are going to need to set:

  1. # Strings - typically set to 1.
  2. Nodes per string - the number of nodes spiralling up the tree.
  3. Spiral wraps - the number of full rotations around the centre the string does between the bottom and top of the tree. For small trees this is likely in the order of 3-6.
