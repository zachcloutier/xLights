Effects with problems

Some effects ... specifically the Video, Pixture and Shader effects will display a solid colour in some error conditions. Knowing what these mean can help you determine what is wrong:

Video Effect

  • Blue - displayed when a video that is shorter than the effect length ends and no looping has been configured ... like an old school VHS recorder.
  • Red - displayed when the video file cannot be opened ... most commonly because it is not where the effect thinks it is. It may also be displayed if the model/buffer lacks the dimensions to display a video.

Picture Effect

  • Red - displayed when the image file cannot be opened ... most commonly because it is not where the effect thinks it is.

Shader Effect

  • Red - displayed when the shader file cannot be opened ... most commonly because it is not where the effect thinks it is.
  • Yellow - displayed when there is an error compiling the shader and thus it is unable to run.
  • Cyan - displayed when your video card has a problem running the shader.
