Getting Assistance

Did you know that on the help menu there are menu items which will take you to:

  • The Official xLight Support Group on Facebook - the primary support forum these days for xLights.
  • The xLights Zoom room which pretty much offers 24x365 support from other members of the user community.
  • The xLights manual
  • The xLights video tutorial archive.
  • The xLights web site.
  • The github repository for reporting issues.
